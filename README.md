# ModernEconomy
# 💰 ModernEconomy — O Coração da sua Economia

> Um sistema de economia robusto, leve e moderno, desenvolvido especialmente para servidores PocketMine-MP (Bedrock Edition).

---

## 📖 Sobre o Projeto
O **ModernEconomy** foi criado para oferecer uma experiência de gerenciamento financeiro intuitiva tanto para administradores quanto para jogadores. Com foco em performance e uma interface limpa, ele permite que você foque no que importa: a diversão do seu servidor.

---

## 🛠️ Funcionalidades e Comandos

<details>
<summary><b>💰 Comandos de Jogador (Clique para abrir)</b></summary>

*   **/money** - Veja o seu saldo atual.
*   **/pay <jogador> <quantia>** - Envie dinheiro para outro jogador de forma segura.
*   **/money help** - Lista todos os comandos disponíveis para o usuário.
</details>

<details>
<summary><b>👑 Comandos de Administrador (Clique para abrir)</b></summary>

*   **/setmoney <jogador> <quantia>** - Define o saldo exato de um jogador.
*   **/addmoney <jogador> <quantia>** - Adiciona uma quantia ao saldo atual.
*   **/takemoney <jogador> <quantia>** - Remove uma quantia do saldo do jogador.
*   **/seemoney <jogador>** - Espia o saldo de qualquer jogador online.
</details>

<details>
<summary><b>🔐 Permissões (Configuração de Grupos)</b></summary>

| Permissão | Descrição | Padrão |
| :--- | :--- | :--- |
| `moderneconomy.command.money` | Permite ver o próprio saldo e usar /pay | Todos |
| `moderneconomy.admin.manage` | Permite usar comandos de set, add e take | OP / Admin |
| `moderneconomy.admin.see` | Permite ver o saldo de outros jogadores | OP / Admin |

</details>

---

## 🚀 Como Instalar e Usar

1.  Baixe o arquivo `.phar` aqui no **Poggit**.
2.  Mova o arquivo para a pasta `/plugins/` do seu servidor PocketMine-MP.
3.  Reinicie o servidor para gerar a pasta de configuração.
4.  Configure o saldo inicial e o símbolo da moeda no arquivo `config.yml`.
5.  **Pronto!** Sua economia já está rodando.
